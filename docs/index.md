# Admiral Provider

The Admiral provider is used to interact with the resources supported by [Admiral](https://admiral.io). The provider needs to be configured with the proper credentials before it can be used.

Use the navigation to the left to read about the available resources and data sources.

## Example Usage

```terraform
terraform {
  required_providers {
    admiral = {
      source  = "admiral/admiral"
      version = "~> 1"
    }
  }
}

provider "admiral" {
  host  = var.admiral_host
  token = var.admiral_token
}
```
